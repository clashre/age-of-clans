# Contribuyendo a Age of Clans

## Guía de Contribución

Gracias por tu interés en contribuir a Age of Clans. Aquí están las directrices:

### Antes de Empezar

1. Fork el repositorio
2. Clona tu fork
3. Crea una rama para tu feature: `git checkout -b feature/nombre-feature`

### Desarrollo

1. Asegúrate de que tu código sigue el estilo del proyecto
2. Prueba tu código localmente
3. Comenta tu código cuando sea necesario

### Submitting

1. Push a tu rama
2. Abre un Pull Request
3. Describe los cambios claramente

### Reportar Bugs

Usa GitHub Issues para reportar bugs. Incluye:
- Descripción clara
- Pasos para reproducir
- Comportamiento esperado vs actual
- Tu entorno (OS, versiones, etc.)

## Código de Conducta

Sé respetuoso y constructivo. Queremos una comunidad acogedora.
